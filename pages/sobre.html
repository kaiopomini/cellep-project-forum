Sobre


O Que é esse site?

Esse site foi desenvolvido com intuito acadêmico para a aplicação das habilidades adquiridas durante o curso de desenvolvimento web front-end da Estação Hack e CEL.LEP.


O que foi utilizado para o desenvolvimento desse site?

Para o desenvolvimento desse site foi utilizado HTML, CSS e Javascript, Bootstrap 4 para a responsividade.

Também foi utilizado a api rickandmortyapi para gerar as imagens e as informações sobre os personagens.
